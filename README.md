# 📜 Projeto Cordel Moderno

Projeto desenvolvido como desafio prático do **Módulo 3** do curso de **HTML5 e CSS3** do [Curso em Vídeo](https://www.cursoemvideo.com/), ministrado pelo professor [Gustavo Guanabara](https://github.com/gustavoguanabara).

O foco deste módulo e deste projeto foi o domínio de **Design Responsivo** e o uso avançado de propriedades de imagem de fundo para criar o efeito **Parallax**.

## 🚀 Tecnologias e Conceitos Aplicados

- **HTML5**: Estruturação semântica.
- **CSS3 Avançado**: 
    - Uso de `background-attachment: fixed` para criar o efeito de rolagem.
    - Implementação de **Media Queries** para garantir a responsividade em diferentes dispositivos.
    - Estilização de textos com sombras e fundos translúcidos (`rgba`).
- **Tipografia**: Integração com Google Fonts (*Passion One* e *Sriracha*).

## 📱 Diferenciais do Projeto

- **Mobile First**: Design pensado para funcionar perfeitamente em telas pequenas antes de expandir para desktops.
- **Efeito Parallax**: As imagens de fundo permanecem estáticas enquanto o texto desliza sobre elas, criando uma experiência visual rica.
- **Tradição e Tecnologia**: O contraste entre o tema clássico da poesia de cordel e as tecnologias modernas de desenvolvimento web.
